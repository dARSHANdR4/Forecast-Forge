# **App Name**: Forecast Forge

## Core Features:

- Data Upload: Simple interface for users to upload their data in CSV format.  Data will be stored in browser memory.
- Model Training: Select a pre-configured model type (Linear Regression) and train it on the uploaded data. Allow users to specify target and feature columns.
- Prediction and Download: Display the trained model's predictions for the training dataset in tabular format and allow the user to download the predictions as a CSV file.  The page will also present some error metrics (MSE).

## Style Guidelines:

- Primary color: Dark blue (#2c3e50) for a professional look.
- Secondary color: Light gray (#ecf0f1) for backgrounds and contrast.
- Accent: Teal (#3498db) for interactive elements and highlights.
- Clean and simple layout with clear sections for data upload, model training, and prediction display.
- Use minimalist icons to represent different actions and data types.
